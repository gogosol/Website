### Typography

Typography should feel:
- executive
- sharp
- confident
- clean
- high-information

Use a hierarchy with:
- strong hero display
- precise section headers
- readable body text
- compact labels and metadata

### Layout behavior
- 12-column grid on desktop
- disciplined spacing
- asymmetrical but balanced sections
- dense information blocks broken into readable modules
- alternating rhythms between text-heavy and visual-heavy sections

### Component style
- subtle borders
- soft glows only where tasteful
- premium cards
- layered panels
- data-table influence where useful
- minimal but sophisticated hover states

### Motion
Use restrained motion only:
- fade and slight rise on scroll reveal
- subtle parallax or background drift if elegant
- card hover microinteractions
- animated connectors/lines only if very tasteful

No flashy over-animation.
